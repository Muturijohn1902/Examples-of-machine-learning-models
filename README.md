# Examples of Machine Learning Models

This repository contains clean, practical, and complete examples of end-to-end Machine Learning projects.  
Each project includes Exploratory Data Analysis (EDA), data preprocessing, model training.
The source files are from kaggle

Currently, the repository includes:

- Car Price Prediction: https://www.kaggle.com/code/rehamfaw/car-price-predictionc/input
- Customer Churn Prediction: https://www.kaggle.com/datasets/blastchar/telco-customer-churn/data
